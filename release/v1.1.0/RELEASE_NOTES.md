# v1.1.0 릴리즈 노트

**릴리즈 일자**: 2026-07-06

IT 및 정보보호 교육 모듈을 대폭 강화한 릴리즈입니다. 문제은행, 퀴즈 풀이 UI 개선, 오답 리뷰, 교육·훈련 결과 화면을 추가합니다.

## 신규 기능

### 문제은행 관리
교육 퀴즈 문항을 코스와 독립적으로 관리하는 문제은행 메뉴를 관리 섹션에 추가합니다.

- 문항 CRUD, 분류·난이도(상/중/하)·키워드 검색 조회
- 엑셀 일괄 업로드로 문항 대량 등록 (템플릿 다운로드 제공)
- 교육 등록 화면에서 **문제은행 가져오기** — 필터·페이지 단위 선택, 난이도별 랜덤 선택 지원
- 기본 샘플 문항 초기 데이터 제공

### 퀴즈 풀이 UI 개선 — 한 문항씩 표시
기존 전체 문항 스크롤 나열 방식을 한 문항씩 표시 방식으로 개편합니다.

- 이전/다음 버튼 이동, 진행률 바, 문항 번호 점프
- 제출 시 미답변 문항 안내

### 퀴즈 오답 리뷰
퀴즈 제출 직후 결과 화면에서 틀린 문항의 리뷰를 제공합니다.

- 문항별로 내가 선택한 답 / 정답을 보기 본문과 함께 표시
- 문항에 해설이 등록된 경우 해설을 함께 표시
- 재응시("다시 도전") 전에 어떤 문제를 왜 틀렸는지 확인 가능

### 교육·훈련 결과 화면
교육 이수 현황과 모의 악성메일 훈련 결과를 탭으로 조회하는 결과 화면을 추가합니다.

## 변경 사항

- 메뉴·타이틀 명칭 변경: '보안 교육' → **'IT 및 정보보호 교육'** (한국어/영어 i18n, RBAC 라벨, README·docs 반영)
- 교육 등록 화면: 문항 리스트 앞/끝 양쪽에 문항 추가·저장 버튼 배치

## 데이터베이스 변경

- `quiz_bank_questions` 테이블 신규 (JPA 자동 생성 — 수동 마이그레이션 불필요)
- `quiz_questions`에 해설(`explanation`) 컬럼 사용

## 업그레이드

```bash
git pull
docker compose up -d --build
```

스키마 변경은 JPA가 자동 반영하므로 수동 마이그레이션이 필요하지 않습니다.
