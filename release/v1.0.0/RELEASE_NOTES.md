# v1.0.0 릴리즈 노트

**릴리즈 일자**: 2026-07-04

KSecPortal 최초 릴리즈입니다. 스타트업·중소기업을 위한 올인원 정보보안 관리 시스템으로, 아래 기능을 포함합니다.

## 주요 기능

### 정보보호 관리체계
- **ISMS-P 증적관리** — 101개 인증항목(관리체계·보호대책·개인정보), 연도별 증적 CRUD, 증적 파일 참조 재사용, CSV/PDF 다운로드, 엑셀 일괄 등록
- **ISMS-P 통제항목-정책 매핑** — 101개 항목 실 DB 연동, 검색 Picker
- **위험평가** — 자산×위협 자동 생성, 연도·차수 관리, 스냅샷 격리, 환경 필터, 결과 엑셀 다운로드
- **위협관리** — 위협 카탈로그(기본 560개 항목), 위험처리계획

### 보안 운영
- **보안 정책** — 정책 문서 CRUD, 버전·승인 워크플로
- **취약점 관리** — NVD API 연동(CVE 자동 입력), 심각도·상태 추적, 엑셀 일괄 등록
- **보안 인시던트** — 8가지 유형, 5단계 상태, 대응 타임라인, PDF 다운로드
- **자산 관리** — ISMS-P 자산식별 기준(기밀성/무결성/가용성 등급, 개인정보 포함 여부 등 33컬럼), 온프레미스+클라우드 통합
- **보안이벤트 관리** — 10종 보안솔루션 연동, 실시간 모니터링
- **월간 보안점검** — 32개 표준 항목, 담당자 지정, 증적 파일 관리

### IT 및 정보보호 교육
- **교육 관리** — 코스 관리, 객관식 퀴즈(한 문항씩 풀이·진행률 표시), 이수율·점수 추적
- **문제은행** — 분류·난이도별 문항 관리, 엑셀 일괄 업로드, 교육 등록 시 필터·난이도별 랜덤 출제
- **모의 악성메일 훈련** — 피싱 템플릿 CRUD, 캠페인 실시, 클릭/열람/신고 추적, 결과 통계
- **교육·훈련 결과** — 교육 이수 현황과 모의훈련 결과 통합 조회

### 개인정보보호
- **수탁사 관리·점검** — 수탁사 CRUD, 연도별 점검(기본 20개 항목), 증적 관리
- **법령준수관리** — 법제처 API 실시간 조문 조회, 법령검토 이력, Excel 보고서

### 거버넌스·플랫폼
- **정보보호위원회·내부감사·보안 결함사항** 관리
- **대시보드** — 위험/취약점/인시던트/ISMS-P 이행률/증적 현황 실시간 위젯, 드래그 앤 드롭
- **보안** — JWT 인증, MFA(TOTP), Okta SSO(OAuth2 PKCE), 로그인 실패 잠금, 세션 타임아웃, RBAC 메뉴별 권한, nginx CSP·HSTS·Rate Limiting
- **운영** — 암호화 백업(AES-256-GCM, Cron 스케줄), 감사 로그, 다국어(한/영), PDF/CSV/Excel 리포트
