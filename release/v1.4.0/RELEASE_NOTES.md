# v1.4.0 릴리즈 노트

**릴리즈 일자**: 2026-07-09

사이드바에 **도움말 메뉴**를 신설하여 사용자 매뉴얼을 웹페이지로 열람할 수 있게 한 릴리즈입니다. 매뉴얼 문서를 현재 버전 기준으로 전면 최신화하고 전체 스크린샷을 갱신했습니다.

## 신규 기능

### 도움말 메뉴 (웹 매뉴얼 뷰어)
사이드바 하단 **환경설정** 아래에 **도움말** 메뉴가 추가되었습니다. `docs/user-manual.md`를 앱 내에서 웹페이지 형태로 렌더링합니다.

- **목차 패널** — 왼쪽 고정 목차에서 섹션 클릭 이동, 현재 읽는 섹션 자동 하이라이트
- **목차 검색** — 키워드로 목차 필터링
- **스크린샷 확대** — 본문 이미지 클릭 시 라이트박스 확대
- **인쇄** — 매뉴얼 전체 인쇄 / PDF 저장
- 부록의 Mermaid ERD 정의는 접힌 상태(펼침 가능)로 표시
- 매뉴얼 원본은 `docs/user-manual.md` 단일 관리 — Docker 빌드 시 `docs/`에서 자동 포함

### 사용자 추가 — 초기 비밀번호 보기 토글
**관리 > 사용자 관리 > 사용자 추가** 모달의 비밀번호 입력란에 눈 아이콘 토글이 추가되어 입력한 초기 비밀번호를 확인할 수 있습니다. (기존 "새 비밀번호" 토글과 동일한 UX)

## 문서

- 사용자 매뉴얼 전면 최신화 — 누락되었던 **IT 및 정보보호 교육**, **로그 통합관리**, **공지사항**, **문제은행 관리**, **감사 로그**, **설정관리(6개 탭)** 섹션 추가
- 메뉴 명칭을 현재 UI 기준으로 정정 (정보보호위원회, 월간 보안점검, 모의 훈련 등)
- 환경설정 섹션을 실제 탭 구조(화면/계정/정보)로 재작성
- 전체 스크린샷 37장을 v1.4.0 화면 기준으로 재캡처 (신규 7장 포함)
- 변경 이력을 v1.0.0(2026-07) 이후 기준으로 정리

## 업그레이드 방법

```bash
git pull
docker compose build frontend
docker compose up -d frontend
```

백엔드 변경은 없습니다 (버전 표기만 갱신).
