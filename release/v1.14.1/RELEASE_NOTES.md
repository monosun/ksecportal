# v1.14.1 릴리즈 노트

**릴리즈 일자**: 2026-07-22

법령준수관리 > 법령검토에서 **업종 공통 법령의 조문이 조회되지 않던 문제**를 수정했습니다.

## 버그 수정

### 업종 공통 법령 조문 조회 실패

법령검토 화면은 각 법령의 조문을 law.go.kr에서 실시간 조회하고, 조회가 실패하면 내장 조문 데이터로 대체 표시합니다. v1.14.0에서 신설한 **업종 공통** 법령 중 일부(정보통신망법 계열, ISMS-P 고시, 개인정보 영향평가 고시 등)에 내장 조문 데이터가 없어, 실시간 조회가 지연·실패할 경우 조문이 비어 보이는 문제가 있었습니다.

- 아래 8개 법령에 내장 조문 데이터(실시간 조회 실패 시 폴백)를 추가했습니다.
  - 정보통신망 이용촉진 및 정보보호 등에 관한 법률 / 시행령 / 시행규칙
  - 정보보호 및 개인정보보호 관리체계 인증 등에 관한 고시(ISMS-P)
  - 정보보호 조치에 관한 지침
  - 개인정보 영향평가에 관한 고시
  - 인공지능 발전과 신뢰 기반 조성 등에 관한 기본법 / 시행령

### 폐지 법령 정리 — 개인정보 보호법 시행규칙 제외

**개인정보 보호법 시행규칙**은 폐지되어 law.go.kr 현행 법령 조회가 되지 않으므로, 업종 공통 목록에서 제외했습니다.

## 데이터베이스 · API
