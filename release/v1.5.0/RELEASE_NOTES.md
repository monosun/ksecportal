# v1.5.0 릴리즈 노트

**릴리즈 일자**: 2026-07-10

문제은행 관리와 교육등록 문제 가져오기 기능을 개선한 릴리즈입니다. 분류별 문제 현황·일괄 삭제를 추가하고, Excel 일괄등록의 중복 문제 유입을 차단했으며, 문제은행에서 문항을 가져올 때 일부 문항이 누락되던 버그를 수정했습니다.

## 신규 기능

### 문제은행 — 분류별 문제 현황·일괄 삭제
**관리 > 문제은행** 목록 상단에 **분류별 문제 현황** 카드가 추가되었습니다.

- 분류별 문제 수를 한눈에 표시 (미분류 문항은 별도 버킷으로 표시)
- 분류를 클릭하면 해당 분류로 목록이 필터링됩니다
- 분류 옆 **휴지통** 아이콘으로 해당 분류의 문제를 한 번에 일괄 삭제 (삭제 후 목록·현황 자동 갱신)

### Excel 일괄등록 — 동일 문제 자동 제외
일괄등록 시 **동일한 문제(문제 텍스트 일치)를 등록하지 않고 건너뜁니다.**

- 이미 등록된 문항, 그리고 업로드 파일 내 중복 문항을 모두 제외
- 결과 화면에 **성공 / 중복 제외 / 실패** 건수를 구분해서 표시
- **등록시작** 버튼은 업로드 시작 후·완료 후 비활성화되어 같은 파일의 중복 등록을 방지 (다른 파일 선택 시 재활성화)

## 버그 수정

### 교육등록 — 문제은행 가져오기 선택 문항 누락
**IT 및 정보보호 교육 > 교육등록 > 문제은행에서 가져오기**에서 여러 문항을 선택해도 문제 텍스트가 같은 문항이 하나만 추가되던 문제를 수정했습니다. 이제 선택한 문항은 문제 텍스트가 같더라도 각각 모두 추가됩니다.

## API 변경

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/quiz-bank/categories/stats` | 분류별 문제 수 (`[{category, count}]`, 미분류는 `category: null`) |
| DELETE | `/quiz-bank/by-category?category=` | 분류 단위 일괄 삭제 (비어 있으면 미분류 삭제), 삭제 건수 반환 |
| POST | `/quiz-bank/bulk` | 응답에 `skippedCount`(중복 제외 건수) 추가, 동일 문제 자동 제외 |

## 문서

- 사용자 매뉴얼 — 문제은행 관리(분류별 현황·삭제, 일괄등록 중복 제외), 교육등록 문제 가져오기 설명 보강
- API 문서 — 문제은행 신규/변경 엔드포인트 반영
