# KSecPortal 메뉴 구성 (좌측 네비게이션)

좌측 사이드바에 표시되는 전체 메뉴와 각 메뉴의 기능입니다.
메뉴는 **RBAC 권한(menuKey)** 에 따라 사용자별로 숨겨질 수 있으며, **관리** 그룹은 ADMIN 역할에만 표시됩니다.

기준: `frontend/src/components/layout/AppLayout.vue` (v1.10.0)

---

## 메인

| 메뉴 | 경로 | 기능 |
|---|---|---|
| 메인 | `/dashboard` | 통합 대시보드. 위험·취약점·인시던트·증적 현황 요약, 공지사항, 법령 개정정보 위젯 표시 |

---

## 대시보드

| 메뉴 | 경로 | 기능 |
|---|---|---|
| 위험현황 | `/dashboard/risks` | 위험평가 결과를 등급·처리방법별로 집계해 현황 조회 |
| 취약점 현황 | `/dashboard/vulns-status` | 취약점 심각도·조치 상태별 통계 조회 |
| 인시던트 현황 | `/dashboard/incidents-status` | 보안 인시던트 발생 건수·유형·처리 상태 통계 조회 |
| ISMS-P 이행률 | `/dashboard/isms-progress` | ISMS-P 통제항목별 이행 진척도 조회 |
| 증적 제출 현황 | `/dashboard/evidence-status` | ISMS-P 증적 자료 제출·미제출 현황 조회 |

## 정보보호 관리체계

| 메뉴 | 경로 | 기능 |
|---|---|---|
| 보안 정책 | `/policies` | 보안 정책·지침 문서 등록, 버전·개정 이력 관리 |
| 자산 관리 | `/assets` | 정보자산 등록·검색·엑셀 일괄등록, 자산유형별 현황, 시점(스냅샷) 이력, SW 자산 SBOM 맵핑 |
| SBOM 관리 | `/sbom` | SW 및 라이브러리(구성요소) 등록, CycloneDX 내보내기/가져오기, 엑셀 일괄등록 |
| 위협 관리 | `/threat-management` | 기본 위협 목록 조회 및 사용자 정의 위협 추가·관리 (위험평가의 기초 데이터) |
| 취약점 관리 | `/vulnerabilities` | 취약점 등록, 심각도 분류, 조치 상태 관리 |
| 위험평가 | `/risk-assessment` | 위험평가 차수 관리, 위험수용 기준 설정, 위험 항목 등록·일괄처리, 결과 다운로드 |
| 위험처리 계획 | `/risk-treatment` | 수용·감소·회피·이전별 위험 처리 계획 수립 및 이행 관리 |
| ISMS-P 통제항목 매핑 | `/isms-mapping` | 위험·조치 항목을 ISMS-P 통제항목에 매핑 |
| ISMS-P 증적관리 | `/isms` | ISMS-P 통제항목 조회 및 증적 자료 등록·관리 |

## 개인정보보호

메뉴 순서는 개인정보 라이프사이클(수집 → 이용·제공 → 위탁 → 보유 → 파기)을 따릅니다.

| 메뉴 | 경로 | 기능 |
|---|---|---|
| 개인정보 처리현황 | `/privacy/processing` | 개인정보 처리업무 등록 — 처리부서·처리목적·개인정보 항목·보유기간·처리근거·처리시스템·업무흐름도·라이프사이클 |
| 개인정보 파일관리 | `/privacy/files` | 개인정보파일대장 — 파일별 담당부서·DB 테이블·개인정보 항목·보유기간, 민감정보·고유식별정보 포함 여부 |
| 개인정보 수집·이용 관리 | `/privacy/consents` | 수집동의서 버전관리 — 필수/선택 구분, 수집 항목, 이용목적, 처리근거, 개정본 생성 |
| 개인정보 제공관리 | `/privacy/provisions` | 제3자 제공·공동이용·국외이전 관리 (위탁과 별도) — 제공받는 자, 제공 항목·목적·근거, 제공 계약 |
| 수탁사 관리 | `/privacy/contractors` | 개인정보 수탁사 등록·일괄등록, 선택 삭제, 개인정보처리방침 URL 이력 관리 |
| 수탁사 점검 | `/privacy/contractor-checks` | 수탁사 점검 수행, 점검 항목별 결과 기록, 건별 점검 이력 조회 |
| 개인정보 보유기간 관리 | `/privacy/retentions` | 보유기간·만료예정일·삭제예정일 추적, 연장사유, 만료 자동알림 설정 |
| 개인정보 파기관리 | `/privacy/disposals` | 파기계획·파기대상·파기승인·파기이력·파기증적. 파기완료 시 연계된 보유기간 항목을 자동으로 파기완료 처리 |
| 개인정보 영향평가(DPIA) | `/privacy/dpia` | 영향평가 대상 선정, 체크리스트, 위험도, 개선계획, 완료보고 |
| 개인정보 유출관리 | `/privacy/breaches` | 유출사고 등록, 신고기한 관리(인지 후 72시간 자동 산정), 정보주체 통지, 조사결과, 재발방지, 관계기관 신고 |
| 정보주체 권리행사 관리 | `/privacy/rights-requests` | 열람·정정·삭제·처리정지·동의철회 요청 접수와 처리결과, SLA 관리(요청일 + 10일 자동 산정) |
| 개인정보 보호조치 관리 | `/privacy/safeguards` | 접근권한 점검·권한회수·암호화 적용현황·접속기록 점검결과·출력물 관리·반출관리·휴면계정 관리 |
| 법령준수관리 | `/privacy/legal-compliance` | 개인정보보호 관련 법령 준수 항목 관리, 법령 개정정보 및 조회기간 설정 |
| 개인정보 현황보고서 | `/privacy/report` | 경영진·ISMS-P 심사용 자동 집계 — 처리현황·파일·수탁사·제3자 제공·보유기간·파기·권리행사·유출사고·법령 준수현황 |

## 관리 (ADMIN 전용)

| 메뉴 | 경로 | 기능 |
|---|---|---|
| 사용자 관리 | `/admin/users` | 사용자 계정 생성·수정·비활성화, 역할 부여, 계정 잠금 관리 |
| 권한관리 | `/admin/rbac` | 역할별 메뉴 접근 권한(읽기/쓰기) 설정 |
| 공지사항 관리 | `/admin/notices` | 공지사항 등록·수정·삭제, 상단 고정 및 활성화 설정 |
| 코드 관리 | `/admin/codes` | 공통 코드 그룹·코드값 관리 (수탁사 점검항목 등 포함) |
| 문제은행 관리 | `/admin/quiz-bank` | 교육 평가 문항 등록·관리 |
| 감사 로그 | `/admin/audit-logs` | 시스템 전체 변경 이력(감사 로그) 조회 |
| 백업 관리 | `/admin/backup` | 데이터베이스 백업 생성·복원·다운로드 |
| 설정관리 | `/admin/settings-management` | 시스템 전역 설정(로고, 메일서버(SMTP), 세션, 연동 등) 관리 |
